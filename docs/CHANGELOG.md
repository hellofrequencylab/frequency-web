# What's new

All notable, human-facing changes to Frequency. Newest first. Written for people,
not machines (per [Keep a Changelog](https://keepachangelog.com)); technical detail
lives in git history. This file is the single source for the public
[What's new](/help/changelog) page, so it auto-updates on every release.

Versioning follows the release tags (`vMAJOR.MINOR.PATCH`). Add changes under
`## [Unreleased]` as you ship them; on release, rename it to the new version + date.
